#  Expense Tracker App

A **responsive Expense Tracker** built with **React** and **Tailwind CSS** that helps users manage their income and expenses efficiently. The app calculates balance dynamically and provides a clear transaction history.

---

##  Live Demo

 **Live Demo:** https://expenses-tracker-nine-gamma.vercel.app/  


---

##  Features

-  Add income and expense transactions  
-  Automatic calculation of **Balance**, **Total Income**, and **Total Expenses**  
-  Real-time UI updates using React state  
-  Clean and responsive UI (mobile + desktop)  
-  Input validation for better UX  
-  Simple and intuitive design  

---

##  Tech Stack

- **React.js** – Component-based UI
- **Tailwind CSS** – Utility-first styling
- **JavaScript (ES6+)**
- **Vite** – Fast development build tool
